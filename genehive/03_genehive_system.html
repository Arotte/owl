System Architecture
Agent Cast
Wallace
orchestrator
Selection pressure, population management, island control. Reads Fossil, decides who mutates, who breeds, when Hermes fires, when to unlock the next eval stage. All strategic decisions.
LLM agent
Mendel
mutator
Edits the genome one targeted change at a time. Invoked via Cursor CLI as a subprocess. Named after the monk who discovered inheritance by obsessively breeding peas.
Cursor CLI
Cupid
crossover
Reads two high-performing parent genomes and breeds a child combining the best of both. The LLM understands why each parent works — classical GAs can't do this.
Cursor CLI
Reaper
evaluator
Runs eval commands, interprets raw output (logs, JSON, text), makes a binary cull-or-keep decision. An LLM agent that understands error profiles and trends, not a simple comparator.
LLM agent
Fossil
historian
Passive log of all experiments: genome hash, phenotype score, eval output, diffs, lineage. Never signals, never decides. A database with a reporting layer. Wallace reads it.
SQLite
Hermes
migrator
Carries the fittest genome between islands. Fires rarely — isolation preserves diversity. Only in island topology. Named after the messenger god.

Eval Schedule — Reaper's Curriculum
Stage 0 · Smoke Test
Does it run? Is the output valid? Binary pass/fail.
trigger next → always
Stage 1 · Proxy Sprint
10–50 curated examples. High noise, fast iteration. Broad exploration.
trigger next → convergence detected by Wallace
Stage 2 · Capability Unlocks
Narrow high-fidelity evals added sequentially. Tests specific capabilities.
trigger next → per-capability convergence
Stage 3 · Blended Pressure
Multi-objective weighted scoring. Pareto frontier tracking.
trigger next → Pareto front stable
Stage 4 · Integration Oracle
Full benchmark. Expensive. Run on top-K survivors only.
trigger next → grand tournament
Stage 5 · Human Loop (optional)
Aesthetics, edge cases, non-automatable dimensions.
Fossil surfaces candidates to human scorer
Vocabulary & Concepts
The Things
Genome — the code repo being optimised. What Mendel edits, what Cupid splices.
Phenotype — the running system. What Reaper evaluates. The expressed result.
p0 — the root commit. The primordial genome. Where everything begins.
The Topology
Island — an isolated population evolving independently. Each has its own Mendel, Cupid, Reaper.
Pangea — conceptually, the moment before islands diverge. The unified state.
Migration — Hermes carries fittest genome across islands. Rare and precious.
The Git Mapping
Mutation → Mendel's commit
Crossover → Cupid's merge commit
Dead end → git revert
Migration → Hermes cherry-pick across islands
Selection → Reaper cull + Wallace tournament
